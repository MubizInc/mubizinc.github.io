Proof of Existence et DApps

Quête

Intégrer un service d'encrage de documents dans Ropsten (Ethereum) sur votre site.

Parcours

  • Utilisation de blockizor.com
  • Copie du code dans une page de votre site

Livrable et notation

  • Envoyé depuis votre adresse e-mail de l'école (@devinci.fr)
  • Commencer le sujet par "[ESILV][TP][5A][IF-X]"
  • Mettre dans le contenu un lien hypertexte vers votre site

Niveau 0 - Equipement

Il n'y a pas d'équipement spécifique nécessaire pour cette quête car vous utiliserez les outils déjà installés pour les précédents TP.
